Login to docker
docker login

Build the image
docker build -t steveyttt/udemynodealpine42 . 

Build with another tag
docker build -t steveyttt/udemynodealpine42:test .

Push up the image
docker image push steveyttt/udemynodealpine42:test
docker image push steveyttt/udemynodealpine42

Run the image
docker container run --detach --rm -p 80:3000 steveyttt/udemynodealpine42

###Tips when creating images
You can find lots of info about images with inspect:
docker inspect image nginx:alpine

###You can see command history of an image too
docker history nginx:alpine

###you can format output in cli:
####This shows all fields you can filter on:
docker history nginx:alpine --format='{{json .}}'
####You can then filter on one command
docker history nginx:alpine --format='{{json .CreatedBy}}'
